How it works

  • GET /api/char - Returns a list of characters from the database. If no data exists in the database, the API should retrieve the data from the free public API, store it in the database, and return it.

    This endpoint was used while the main page was loading.

  • GET /api/char/id - Returns a single character by ID from the database.

    This endpoint can be used by clicking on the concrete character tab.

  • POST /api/posts - Adds a new character to the database.

    This endpoint can be used by clicking on the "Create character" button. Creation form can be automaticaly filled by clicking on the "magic stick" button.
